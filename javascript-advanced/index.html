FUNCTION

  • Declaration: Functions in JavaScript can be declared using the function keyword.

  • Parameters and Arguments: unctions can take parameters, which act as placeholders for values to be passed during invocation. Arguments are the actual values passed to the function.

  • Return Statement: Functions can use the return statement to send a value back to the calling code. This allows functions to produce a result or perform an action.
