You will use the javascript file to continually change the size of the header, but first we need to learn how to use JavaScript to actually grab the HTML objects from the DOM.

Here are some important documetns attributes:

  • document.URL -- This is the actual URL of the website
  • document.body -- This is everything iside of the body tag
  • document.head -- This will display everything in the head of the page
  • document.links -- These are all the links on the page.

These are the methods we can use to grab the HTML Elements

  • document.getElementById() -- Returns the element with the id
  • document.getElementsByClassName() -- returns list of all elements belongign to a class
  • document.getElementsByTagName() -- retunrs list of all elements with the tag
  • document.querySelector() -- returns the first object matching the CSS style selector
  • document.querySelectorAll() -- returns all objects matching the CSS style selector.
