Welcome to your First Exercise!

For this exercise we will be creating a weight metric converter

We will take in a number in pounds (lbs) and convert it to kilogram (kg). Right now we only know enought to do this one way, but later on we will learn how to add logic to this sort of problems!

To complete this exercise do the following:

  • Link this HTML file to your own .js file
  • Use Javascript to accept a number input in pounds (lbs).
  • Then in your js file cpmvert this number in kilograms (*0.454).
  • Afterwards report back in an alert what the weight in kg.
  • THen log/write "Conversion Completed" to the console.
